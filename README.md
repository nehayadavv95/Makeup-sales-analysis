 Global Makeup Sales Analysis Dashboard

 Project Overview

This project analyzes global makeup sales data to uncover insights related to revenue, customer behavior, product performance, and sales channels.
The goal is to transform raw data into meaningful business insights using data analysis and visualization techniques.



 Dataset Description

The dataset includes the following key fields:

* Sale_ID
* Date
* Brand
* Product_Type
* Country
* Sales_Channel
* Payment_Method
* Price_USD
* Units_Sold
* Revenue_USD



 Tools & Technologies Used

* Power BI (Data Visualization & Dashboard)
* Microsoft Excel (Data Cleaning)
* SQL (Data Analysis - optional)



 Key Performance Indicators (KPIs)

* Total Revenue
* Total Units Sold
* Total Transactions
* Average Revenue per Sale



 Dashboard Features

* Monthly Revenue Trend Analysis
* Country-wise Sales Performance
* Brand Performance Analysis
* Product Category Insights
* Sales Channel Comparison (Online vs Offline)
* Payment Method Distribution


 Key Insights

* A few countries contribute the majority of total revenue.
* Online sales channel dominates overall performance.
* Certain product categories generate higher sales volume.
* Digital payment methods are more frequently used than traditional methods.
* Sales show monthly fluctuations indicating seasonal demand trends.
